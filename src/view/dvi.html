Реализация Multiplayer Perceptron

О программе

В данном проекте реализована простейшая искусственная нейронная сеть в виде перцептрона, которая может быть обучена на открытом наборе данных и осуществлять распознавание 26 рукописных букв латинского алфавита.

Как работать с приложением

Если хотите получить предсказанную букву, используя готовые веса, просто нарисуйте букву и нажмите "click to predict word".

Если хотите тренировать модель, выберите файл с тренировочным датасетом (нажмите "training file") и место чтобы сохранить веса (нажмите "saving file"), далее нажмите "start training".

Если хотите тестить модель, выберите файл с тестовым датасетом (нажмите "testing file") и место чтобы сохранить веса (нажмите "saving file"), далее нажмите "start testing".

Запустить кросс-валидацию "start cross-val".

"epoch count" - менять количество эпох, "learning rate" - скорость обучения, "layer count" - количество слоев, "matrix view/graph view" - переключатель на графовое и матричное представление, "count cross" - количество делений для кросс-валидации.

Установка

Для того, чтобы запустить проект вам нужно использовать команду make install.

Удаление

Для удаления программы вы можете использовать команду make uninstall.

Запуск тестов

Для запуска тестов make test

Архивация

Для архивации make dist
